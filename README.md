# Portfolio Website

A modern portfolio website with Node.js and Python backend options.

## Features

- Responsive portfolio design
- Contact form with Telegram integration
- IP/Location tracking for messages
- DDoS & security protection
- Rate limiting

## Quick Start

### Frontend

Open `index.html` in browser or use Live Server.

### Node.js Backend

```bash
cd backend
npm install
node server.js
# Runs on http://localhost:3000
```

### Python Backend

```bash
cd backend/python
pip install -r requirements.txt
python server.py
# Runs on http://localhost:5000
```

## Security Features

- Rate limiting (5 contact requests/hour)
- XSS protection
- Input sanitization
- Spam detection
- Helmet security headers
